# I ❤️ Youta

A simple animated heart drawing application using HTML5 Canvas and vanilla JavaScript.

## Features

- **Heart Animation**: Draws a heart shape using a mathematical formula, similar to Python's Turtle graphics.
- **Interactive**: Click anywhere to create a ripple effect.
- **Responsive**: Centers the heart on the screen.

## Running Locally

1. Clone the repository.
2. Open `index.html` in your browser (use a local server if needed due to ES modules).

   ```bash
   python3 -m http.server
   ```
